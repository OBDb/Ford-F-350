# Ford F-350

This repository contains signal set configurations for the Ford F-350, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Ford F-350.

The Ford F-350 Super Duty is a heavy-duty pickup truck produced by Ford Motor Company since the 1999 model year. Part of the F-Series Super Duty lineup, it shares its dedicated heavy-duty truck platform with the F-250, F-450, and F-550. The F-350 is available in both single rear wheel (SRW) and dual rear wheel (DRW) configurations, offering class-leading towing and payload capacities for commercial and personal use.
